Core Analysis

  • 01.

    Automated Hour Attribution: Impressions recorded at 00:00:00 - 00:59:59 (H+1) have been merged into the actual scheduled date (H) to align with PO expectations.

  • 02.

    Significant Over-delivery: Tribunnews and Detik.com showed substantial organic traffic spikes during the mid-month period, resulting in higher technical distribution fees.

  • 03.

    Fixed vs Variable: While Media Buy is fixed (CPD), CM360 Serving Fee is variable based on traffic volume. High performance directly correlates with the invoice variance.

Planner Recommendations

A. PO Revision: Proceed with PO adjustment to IDR 16,382,137 to match the mandatory technical invoice from Google CM360.

B. Future Buffer: Implement a minimum 12% ad serving fee buffer for high-traffic CPD placements (Tribun/Detik) to avoid future overspend alerts.

C. Impression Capping: If budget is strictly capped, request publishers to set an "Impression Limit" in the DSP/SSP settings before the campaign starts.
