Tutorial Campaign CPD

Tahap 1: Bikin Taxonomy Otomatis

1

Cari menu custom di atas

Klik menu 🤖 CM360 Tools di sebelah kanan menu Bantuan/Help.

2

Generate Data

Pilih ⚡ 1. Generate & Sync Taxonomy.

3

Masukkan PO Number (Jika Diminta)

Kalau muncul popup, ketik nomor PO-nya. Kalau PO belum turun, kosongin aja lalu klik OK.

WAJIB DIISI: Kolom Kuning!

Setelah script selesai, cek kolom Creative ID dan LP ID. Kalau warnanya KUNING TERANG, artinya datanya kosong. Kamu wajib isi ID ini manual sebelum lanjut ke Tahap 2.

Tahap 2: Export ke CM360

1

Klik menu Export

Kembali ke menu 🤖 CM360 Tools > Pilih 📥 2. Generate Format CM360 Import.

2

Download File Excel

Tunggu loading selesai, nanti muncul popup berisi link hijau. Klik link tersebut, lalu di file baru klik File > Download > Microsoft Excel (.xlsx).

Tutorial Campaign OTT

Secara fungsi sama persis dengan CPD, bedanya menu ini khusus membaca mapping Publisher OTT dan tipe tracking.

Step 1: Generate

  • Klik menu 🎬 OTT/Tracker Tools
  • Pilih Generate & Sync Taxonomy OTT
  • Masukkan PO Number bila diminta
  • Jangan lupa isi ID di kolom kuning!

Step 2: Export

  • Klik menu 🎬 OTT/Tracker Tools
  • Pilih Generate Format CM360 Import
  • Pilih Campaign yang mau diexport (jika ada pilihan)
  • Buka link dan Download Excel-nya

Cheat Sheet & Trik Cepat

Cara Ganti PO Number yang Salah

Gak perlu hapus semua data. Cukup hapus teks PO di kolom Campaign Name (sisakan namanya saja), lalu jalankan menu Generate lagi. Popup PO akan muncul lagi!

Biar Nggak Capek Input ID

Pastikan kamu udah nge-paste ID dari CM360 ke baris sheet yang benar sebelum klik Export. Kalau di-export pas ID masih kosong, hasil Excel-nya bakal ada tulisan [KOSONG_ISI_DULU].

Error "Publisher Belum Terdaftar"?

Itu karena nama pub di sheet beda/typo dengan di sistem (misal: "Detik" ditulis "Dtik"). Benerin namanya di sheet, atau lapor ke PIC buat daftarin nama pub baru ke dalam Code (Master Mapping).
